Northcoders News Backend API Documentation

A RESTful API for articles, topics, users, and comments.

Navigation

  • GET /api/topics
  • GET /api/articles
  • GET /api/users
  • GET /api/articles/:article_id
  • GET /api/articles/:article_id/comments
  • POST /api/articles/:article_id/comments
  • PATCH /api/articles/:article_id
  • DELETE /api/comments/:comment_id

GET /api/topics

Fetches all the topics and returns an object with all.

Example Response:

{
  topics: [
    { slug: "coding", description: "Code is love, code is life" },
    { slug: "football", description: "Footie!" },
    ...
  ]
}

GET /api/article:article_id

Fetches an article by the given article_id and returns an object containing the full article

Example Request:

GET /api/articles/1

Example Response:

{
  article: [
    {
    article_id: 1,
    title: "Running a Node App",
    topic: "coding",
    author: "jessjelly",
    body: "This is part two of a series on how to get up and running with Node.js...",
    created_at: "2020-11-07T00:00:00.000Z",
    votes: 0,
    comment_count: 8
    }
  ]
}

POST /api/article:article_id/comments

Posts a comment for a specified artile_id and returns with the new comment. Votes and created_at are defaulted to 0 and current timestamp if not given.

Example Request:

POST /api/articles/3/comments  { username = "butter_bridge", body = "This is a test body" }

Example Response:

{
  comment: [
    {
    article_title:"Eight pug gifs that remind me of mitch"
    username: "butter_bridge",
    body: "This is a test body",
    votes: 16,
    author: "butter_bridge",
    created_at: "2025-07-29T15:30:00.000Z"
    }, 
  ]
}

PATCH /api/article:article_id

Amends the votes of an article with the given article_id and return an object of the updated article. All the other properties remain the same.

Example Request:

PATCH /api/article/4 { inc_votes: 5 }

Example Response:

{
  article: [
    {
    title: "Student SUES Mitch!",
    topic: "mitch",
    author: "rogersop",
    body: "We all love Mitch and his wonderful, unique typing style. However, the volume of his typing has ALLEGEDLY burst another students eardrums, and they are now suing for damages",
    created_at: 1588731240000,
    votes: 5,
    article_img_url:
      "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700",
    }, 
  ]
}

DELETE /api/comments/:comment_id

Deleted the full comment and all its properties of the given comment_id and responds with an empty object.

Example Request:

DELETE /api/comments/7

{ }
